Texdoc

Welcome. This is the homepage of the texdoc project. At least, it will be, as soon as I write it :-)

For now, a few bits of information. The relevant mailing list to report bugs, request features, etc. is texdoc@tug.org. You can subscribe to the list to follow texdoc news and development, but you don't need to be subscribed to participate.

Known bugs

  • Problems with recent versions of Adobe Reader and Adobe Acrobat have been reported on Windows. Though the problem is not fully understood yet, a work around is known. It will be available with the regular updates via tlmgr from 2008/09/12 (maybe 13 depending on the mirror's sync). You can also get it from this zip file, to be unzipped in TLROOT\bin\win32.
  • The original documentation, help message and man page point to the TeX Live mailing list, because the separate texdoc mailing list didn't exist when TL2008 was published. Now it is better to use the texdoc mailing list instead.
