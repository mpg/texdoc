Texdoc

Texdoc helps you find and view documentaton in TeX Live. Its primary function is to locate relevant documentation for a given keyword (typically, a LaTeX package name) on your hard disk, and open it in an appropriate viewer.

You can browse texdoc's documentation online: user manual and man page.

Texdoc is part of the TeX Live distribution. (A texdoc command is available in MikTeX, but it is actually a shortcut for another program: mthelp, developped independantly.)

Mailing list

There is a texdoc mailing list for you to ask questions, report bugs, suggest more features, offer your help, etc. You don't need to subscribe to the list in order to post a message and receive the answers.

Development version

Texdoc's development is hosted on GitHub. Instructions for getting and using it on Unix and Windows are provided in a separate page.
