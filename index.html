Texdoc

Find and view documentation in TeX Live

Texdoc's goal is to make it easier to consult TeX related documentation. Its primary function is to locate relevant documentation for a given keyword (typically, a LaTeX package name) on your hard disk, and open it in an appropriate viewer.

It is part of the TeX Live distribution. (A texdoc command is available in MikTeX, but it is actually a shortcut for another program: mthelp, developped independantly.)

Documentation

The documentation for the current version in TeX Live is available online: user manual and man page.

Mailing list

Questions, feature reports and bug requests (wait... maybe it's the other way round) are welcome on the texdoc mailing list. Please note that you don't need to subscribe to the list in order to post a question and receive the answers.

Development version

Texdoc's development is hosted on GitHub. Instruction for getting and using it on Unix and Windows are provided in a separate page.

Known problems

  • Problems have been reported when trying to open pdf files with Adobe Reader 5.0.5 under windows when AR is already running.
  • Using TeX Live 2008 (DVD release), there was problems with recent versions of Adobe Reader on windows. They have been fixed in the tlnet update of 2008/09/12, by some changes to the TL wrapper scripts. Please upgrade bin-texdoc using tlmgr if you run into this problem.
