Texdoc - svn version

Getting it

svn co svn://tug.org/texdoc/trunk/

Running it

The command - Unix

Just symlink trunk/texdoc.tlu to ~/bin/texdoc (where trunk/ is your local checkout, and ~/bin is assumed to be in your PATH before the TeX Live binaries).

The command - Windows

Here things are a bit different, since there are generally no symlinks, so texdoc.bat is actually a script calling texdoc.tlu after locating it in the texmf trees. So, just put a copy of trunk/texdoc.tlu in TEXMFHOME/scripts/texdoc (you may need to create this directory).

Configuration file

Since the name or meaning of some options used in the default configuration file may change, you need to use the svn version of texdoc.cnf too. Install a copy (or symlink) of it as TEXMFHOME/texdoc/texdoc-dist.cnf (notice the -dist part). It will completely override the configuration files in TEXMFDIST and TEXMFLOCAL.

If everything is installed correctly, running texdoc -f should yield something similar to the following.

texdoc version: 0.4+ svn r33
Configuration files are:
    absent /home/mpg/texmf/texdoc/texdoc-linux-latest.cnf
(*) active /home/mpg/texmf/texdoc/texdoc.cnf
    active /home/mpg/texmf/texdoc/texdoc-dist.cnf
    absent /usr/local/texlive/texmf-local/texdoc/texdoc-linux-latest.cnf
    absent /usr/local/texlive/texmf-local/texdoc/texdoc.cnf
    disabled /usr/local/texlive/2008/texmf/texdoc/texdoc.cnf
(*) This is the recommended configuration file for your personal preferences.
